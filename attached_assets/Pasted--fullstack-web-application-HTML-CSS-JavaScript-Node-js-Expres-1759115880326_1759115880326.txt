Φτιάξε ένα πλήρες fullstack web application χρησιμοποιώντας HTML, CSS, JavaScript και Node.js/Express, με την ακόλουθη λειτουργικότητα:

1. **Login/Authentication**:  
   - Η εφαρμογή ξεκινά με φόρμα login.  
   - Ο χρήστης εισάγει username και password.  
   - Μετά το login, το σύστημα ρωτά αν είναι **Tutor** ή **Student**.  
   - Διαχείριση χρηστών γίνεται με βάση δεδομένων (μπορεί να είναι Replit DB ή SQLite).

2. **Επιλογή μαθήματος**:  
   - Μετά τον ρόλο, εμφανίζονται τα διαθέσιμα μαθήματα: π.χ. Math, Computer Programming, Language Learning, και άλλα που θα προσθέσω αργότερα.  
   - Ο χρήστης επιλέγει μάθημα.

3. **Επιλογή επιπέδου και κεφαλαίου**:  
   - Μετά την επιλογή μαθήματος, ο χρήστης επιλέγει επίπεδο (π.χ. Beginner, Intermediate, Advanced).  
   - Στη συνέχεια, εμφανίζονται τα διαθέσιμα κεφάλαια του επιπέδου.  
   - Στο **πάνω αριστερά μέρος της σελίδας** εμφανίζονται τα υποκεφάλαια του επιλεγμένου κεφαλαίου.

4. **Main περιεχόμενο (κεντρικό κομμάτι της σελίδας)**:  
   - Το κύριο περιεχόμενο της σελίδας είναι το **interactive μάθημα**.  
   - Κάποιες σελίδες περιλαμβάνουν μόνο θεωρία, άλλες μόνο ασκήσεις.  
   - Οι ασκήσεις είναι **multiple choice** ή **drag-and-drop**.  
   - Το περιεχόμενο αλλάζει **χωρίς να φορτώνει ξανά ολόκληρη η σελίδα** (SPA behavior, dynamic rendering με JavaScript / React / Vue).

5. **Notepad για καθηγητή**:  
   - Στη **δεξιά πλευρά της σελίδας** υπάρχει notepad.  
   - Ο καθηγητής μπορεί να γράφει/σχεδιάζει με **ποντίκι, stylus ή πληκτρολόγιο**.  
   - Τα σχέδια/σημειώσεις ενημερώνονται σε πραγματικό χρόνο (ή αποθηκεύονται για το session).

6. **Reactive/Single Page Application**:  
   - Όλες οι αλλαγές στο περιεχόμενο της σελίδας γίνονται **χωρίς reload**.  
   - Πλοήγηση μεταξύ σελίδων/ασκήσεων γίνεται δυναμικά.

7. **Διαδραστικότητα & μέλλον**:  
   - Ο καθηγητής μπορεί να προσθέτει νέα μαθήματα, κεφάλαια, υποκεφάλαια.  
   - Το σύστημα να είναι επεκτάσιμο για future lessons σε προγραμματισμό, γλώσσες, κλπ.

**Στόχος:** Δημιουργία web εφαρμογής τύπου **reactive tutoring platform** που τρέχει πλήρως από browser, χωρίς downloads, με interactive μαθήματα, exercises και notepad για καθηγητή, έτοιμο για επέκταση σε πολλά μαθήματα και επίπεδα.

Πρόσθετα:  
- Χρησιμοποίησε σύγχρονες τεχνολογίες για SPA (π.χ. React ή Vue).  
- Χρησιμοποίησε CSS για responsive design.  
- Χρησιμοποίησε Node.js/Express για backend και DB για users & lessons.  
- Προετοίμασε τη δομή ώστε να μπορεί να προστεθούν νέα μαθήματα/επίπεδα εύκολα.  
- Αν είναι δυνατό, παράδειγμα με **ένα πλήρες μάθημα στα Μαθηματικά**, με θεωρία και ασκήσεις multiple choice/drag-and-drop.
